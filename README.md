# gamepass-portal

Portal do cliente para GamePass Store.

## Setup

```bash
npm install
npm run dev
```

## Configuração

Crie um arquivo `.env.local` com:

```env
VITE_SUPABASE_URL=https://tbnafolszkclonzxjivy.supabase.co
VITE_SUPABASE_ANON_KEY=your_key_here
```

## Funcionalidades

- Login/Registro de clientes
- Visualização de contas GamePass
- Histórico de compras
- Renovação via WhatsApp

## Deploy

```bash
npm run build
```
